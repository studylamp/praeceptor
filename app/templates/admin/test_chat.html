Chat test

A faithful dry run of the student experience: the gate decides whether a message is on-topic (just as it would for the student), then the tutor replies, and the conversation is kept so you can ask follow-ups. Nothing is saved and it doesn't count toward caps. Use “New conversation” to start fresh. Good for checking rich output (LaTeX, SVG diagrams) and how the gate and tutor behave.

Debug log

One record per turn: the gate verdict and, for an answered turn, the tutor’s finish reason, token counts, and a partial flag (true if the stream broke mid-reply — the classic cause of a cut-off answer). Cleared by “New conversation.”
